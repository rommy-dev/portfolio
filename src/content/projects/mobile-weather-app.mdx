## Contexte du projet

Une application mobile simple pour consulter la meteo locale et recevoir des alertes. Le but etait de proposer un usage rapide, sans friction, avec une interface claire.

---

## Fonctionnalites principales

- **Geolocalisation automatique** et recherche manuelle
- **Previsions horaires et journalières**
- **Notifications** personnalisées selon la meteo
- **Mode hors ligne** pour les dernières données

---

## Architecture technique

### Mobile — React Native + TypeScript

- Composants cross-platform pour iOS et Android
- Gestion d'etat avec Redux
- Theme simple et efficace

### Donnees — OpenWeather API

- Caching des reponses pour limiter les appels
- Gestion des erreurs reseau

---

## Defis techniques rencontres

- **Permissions de localisation** variables selon les OS
- **Performance** sur devices modestes
- **Stabilite** des appels API
